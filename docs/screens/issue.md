# Screen: Issue

**Route:** `/<teams|topic|org|space>/{id}/issues/{issue_id}`

## Purpose

The detail page for a single issue. Shows the issue's title, description, author (unless anonymous), status, assignee, deadline, progress, vote totals, and supporter list (subject to attribution rules).
